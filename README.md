# Sentimenti
A company sentiment search engine


# How to run BE
- Install Python
- Install the dependecies using pip

```
pip install newsapi
pip install nltk
pip install django
```


- Navigate to the BE root folder in the terminal

```cd BE```

- Run:

``` python manage.py runserver ```

# How to run FE
- Have npm installed
- Navigate to the FE root folder in the terminal

```cd FE```

- Run:

```npm start build```
